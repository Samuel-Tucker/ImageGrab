How it fits

Keep the current ImageGrab rhythm: capture first, decide what to do next, then share either the image or the text.

1

Capture

Use Ctrl+Cmd+G or Opt+Cmd+G exactly as today. No new capture mode is required for the first version.

2

Detect

Run OCR on the preview image or selected capture when the user chooses Copy Text.

3

Review

Show the detected text in a compact result panel before it silently replaces the clipboard.

4

Paste

Copy clean text for bug reports, prompts, docs, emails, issue comments, and terminal notes.

First useful version

The win is speed. Avoid settings, cloud sync, and library indexing until the one-click OCR loop feels excellent.

Why this beats the other upgrades first

More useful than search alone Search gets stronger after OCR text exists.
Lower risk than scrolling capture No stitching, browser automation, or fragile scroll state.
Complements annotations Text extraction helps explain what the arrows and boxes are pointing at.
Developer-heavy value Errors, dialogs, logs, screenshots, and UI labels become prompt-ready context.
